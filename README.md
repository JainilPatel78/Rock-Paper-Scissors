# Rock-Paper-Scissors ✂️📄🪨

A classic, interactive web-based game built to showcase fundamental JavaScript logic and DOM manipulation. This project features a fully functional user vs. computer gameplay loop with instant visual feedback and persistent score tracking during the session.

## 🚀 Key Features

* **Interactive Gameplay:** Users can select their move, triggering the computer to make a randomized selection in real-time.
* **Core Game Logic:** Implemented standard game rules using JavaScript conditional statements to determine wins, losses, and ties.
* **Score Tracking:** A dynamic scoreboard that instantly updates the user and computer scores after each round.
* **Dynamic UI Updates:** Utilized JavaScript DOM manipulation to update the interface with the current round's results and visual feedback without reloading the page.

## 💻 Tech Stack

* **Frontend Logic:** JavaScript (Vanilla)
* **Markup:** HTML5
* **Styling:** CSS3
